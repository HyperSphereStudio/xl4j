# General Notes
## Excel Types
| Excel Type   | Notes/Values |
| Number       | All floating point, note that it's not IEE-compliant.  In particular NaN -> #NUM!, |
|              | Inf -> #NUM! and subnormals are truncated to 0.  The range is 1E-307 <= \| x \| < 1E308 |
| Boolean      | TRUE or FALSE |
